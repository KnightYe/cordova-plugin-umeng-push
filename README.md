# cordova-plugin-umeng-push

A cordova plugin, a JS version of Umeng Push SDK

# Install

1. ```cordova plugin add https://github.com/hkizuna/cordova-plugin-umeng-push  --variable umengpushappid=your_umengpushappid```

2. ```cordova build ios```
